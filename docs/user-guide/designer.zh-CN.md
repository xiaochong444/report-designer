# 设计器

设计器包提供可视化的报表编辑界面。本指南涵盖其 UI 布局、组件工作流和关键交互。

## Designer 组件

`Designer` 组件是顶层入口：

```tsx
import { Designer } from '@report-designer/designer';
import type { ReportTemplate } from '@report-designer/core';

function App({ template, data }: { template: ReportTemplate; data: unknown }) {
  return (
    <Designer
      template={template}
      data={data}
      onTemplateChange={(updated) => { /* 保存或使用更新后的模板 */ }}
    />
  );
}
```

### Props

| 属性 | 类型 | 说明 |
| --- | --- | --- |
| `template` | `ReportTemplate` | 当前报表模板。 |
| `data` | `unknown` | 运行时 JSON 对象或数组。字段和行集会自动推断。 |
| `onTemplateChange` | `(template: ReportTemplate) => void` | 模板修改时调用。 |

### 可组合的 Shell 组件

对于高级布局，你可以单独组合 shell 的各部分：

```tsx
import { DesignerShell, DesignerStatusBar, DesignerRibbon, DesignerLeftPanel, DesignerPropertyPanel, DesignerCanvasFrame } from '@report-designer/designer';
```

- `DesignerShell` —— 外层布局容器。
- `DesignerRibbon` —— 顶部工具栏，包含标签页（主页、插入、数据、视图）。
- `DesignerLeftPanel` —— 报表树和组件面板。
- `DesignerPropertyPanel` —— 上下文敏感的属性编辑器。
- `DesignerCanvasFrame` —— 带标尺、网格和缩放控制的画布。
- `DesignerStatusBar` —— 状态栏，显示缩放、模式和坐标。

## UI 布局

### 功能区工具栏

功能区按标签页组织：

- **主页** —— 字体格式化（字体、字号、加粗、斜体、下划线、对齐）、边框、文本样式、撤销/重做、剪贴板（复制/剪切/粘贴）。
- **插入** —— 添加组件（文本、图片、表格、图表、条码、二维码、复选框、页码、日期时间、线条、形状、子报表、面板、富文本）。
- **数据** —— 根据示例 JSON 推断数据字典。
- **视图** —— 缩放控制、模式切换（设计/预览）、页面设置对话框。
- **页面** —— 页面方向、边距、尺寸、水印、边框设置。

### 报表树（左侧面板）

显示层级结构：

```
Report
  └── Page 1
        ├── Report Title
        ├── Page Header
        ├── Header
        ├── Data Band (items)
        ├── Footer
        └── Page Footer
```

你可以：

- 选择带或组件来导航画布。
- 拖拽重新排列带的顺序。
- 右键点击弹出上下文菜单（添加带、删除、复制）。

### 带区类型

带区决定一段横向内容在什么时候输出、如何重复。典型业务单据通常用静态带区放标题和页面固定内容，用数据带输出明细行。

| 带区 | 作用 | 常见内容 |
| --- | --- | --- |
| `reportTitle` | 报表开始时输出一次。 | 单据标题、公司名称、单号、开头摘要。 |
| `pageHeader` | 每页顶部重复输出。 | Logo、页标题、打印日期、每页都要出现的固定列标题。 |
| `header` | 数据区域之前输出。 | 区域标题、主记录字段，如订单号、客户、仓库、经办人。 |
| `columnHeader` | 多列或表格式明细前输出。 | 品名、数量、单价、金额等表头。 |
| `groupHeader` | 分组表达式变化时输出。 | 客户分组、品类分组、日期分组、仓库分组。 |
| `data` | 遍历 `root`、`items`、`orders.items` 等行集。 | 明细行、重复文本字段、明细表格、逐项条码。 |
| `hierarchicalData` | 从自动推断的数组路径输出树形行，并按 `childrenField` 展开子行。 | 组织架构、品类树、BOM 结构。 |
| `groupFooter` | 每个分组结束后输出。 | 分组合计、分组数量、分组备注。 |
| `columnFooter` | 列/明细区域结束后输出。 | 列合计、表格区域收尾行。 |
| `footer` | 数据区域之后输出。 | 区域合计、签名栏、审批说明。 |
| `pageFooter` | 每页底部重复输出。 | 页码、打印时间、保密说明。 |
| `reportSummary` | 报表最后输出一次。 | 总合计、最终备注、结尾签名。 |
| `overlay` | 作为页面覆盖层内容输出。 | 类水印标记、套打参考、背景标签。 |

大多数单据可以按这个结构组织：`reportTitle` 放标题，`header` 放主记录字段，`data` 或 `hierarchicalData` 放明细，`footer` 放合计，`pageFooter` 放页码。只有需要每页重复的内容才放到页面级带区。

### 画布

画布是核心编辑区域：

- 组件使用 x/y 坐标定位（单位为 mm）。
- 带显示为水平条带。
- 顶部和左侧的标尺显示尺寸。
- 使用鼠标滚轮或缩放控制进行放大/缩小。
- 选中的组件显示缩放手柄。
- 按住 Shift+点击或拖拽矩形进行多选。

### 属性面板

右侧属性面板显示当前选中项的可编辑属性：

- **带属性** —— 高度、可见性、打印设置、数据绑定、行为选项。
- **组件属性** —— 位置、大小、字体、边框、背景、文本内容、格式化、事件。
- **表格属性** —— 行/列数、单元格编辑、绑定模式、边框设置。
- **图表属性** —— 图表类型、数据绑定、标题、图例、坐标轴、标签、主题。

## 组件面板

面板列出可用的组件类型，按类别组织：

- **文本组件** —— 文本、富文本、页码、日期时间
- **可视化组件** —— 图片、条码、二维码、复选框
- **布局组件** —— 面板、线条、形状
- **数据组件** —— 表格、图表
- **高级** —— 子报表

从面板拖拽组件到画布上，或使用"插入"功能区添加到当前带的中心。

## 设计工作流程

### 1. 设置页面

1. 从"视图"标签页打开**页面设置**对话框。
2. 配置纸张尺寸（A4、Letter、自定义）、方向（纵向/横向）和边距。
3. 可选设置水印和页面边框。

### 2. 提供示例数据

1. 通过 `data` 属性传入有代表性的 JSON。
2. 或进入**数据**标签页 → **数据字典**，粘贴示例 JSON。
3. 设计器会自动推断 `root` 字段树和嵌套数组路径。

### 3. 添加带

1. 使用"插入"功能区 → **带** → 选择带类型。
2. 点击新带应该插入在哪个带之后。
3. 常见布局：`reportTitle` → `pageHeader` → `header` → `data` → `footer` → `pageFooter`。

### 4. 添加组件

1. 在画布或报表树中选择目标带。
2. 从面板拖拽组件或使用"插入" → 组件类型。
3. 在画布上定位和调整组件大小。
4. 在属性面板中设置其属性。

### 5. 绑定数据到组件

- **文本组件**：将 `text` 属性设置为 `{customer.name}` 或 `{items.product.name}` 这样的表达式。
- **表格组件**：将绑定模式设置为 `detail`，选择 `items` 这样的数组路径，并将列映射到字段。
- **图表组件**：基于 `items` 这样的推断行集配置维度和度量。

### 6. 应用样式

- 选中文本组件，然后从**文本样式库**中选择样式。
- 通过样式库对话框创建自定义样式。
- 样式控制字体、颜色、对齐、边框和背景。

### 7. 预览

切换到预览模式查看渲染结果。分页、分组和表达式都会被计算。

## 键盘快捷键

| 快捷键 | 操作 |
| --- | --- |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` | 重做 |
| `Ctrl+C` | 复制选中的组件 |
| `Ctrl+X` | 剪切选中的组件 |
| `Ctrl+V` | 粘贴剪贴板 |
| `Delete` | 删除选中的组件或带 |
| `方向键` | 微移选中的组件（1px） |
| `Shift+方向键` | 微移选中的组件（10px） |
| `Ctrl+D` | 复制选中的组件 |
| `Ctrl+B` | 切换选中文本的加粗 |
| `Ctrl+I` | 切换选中文本的斜体 |
| `Ctrl+U` | 切换选中文本的下划线 |

## 文本样式库

文本样式库（通过功能区工具栏打开）允许你：

- 创建、编辑、复制、重命名和删除文本样式。
- 设置一个样式为**默认**（新文本组件自动使用它）。
- 查看每个样式被多少组件引用。
- 将样式应用到选中的组件。
- 从组件取消绑定样式。

## 条件格式库

条件格式库（通过功能区工具栏打开）允许你：

- 创建命名条件格式规则。
- 定义带表达式、运算符和样式覆盖的规则。
- 将格式应用到选中的组件。
- 管理格式范围和优先级。

## 国际化

设计器支持中文和英文界面。使用 `DesignerI18nProvider` 配置语言环境：

```tsx
import { DesignerI18nProvider } from '@report-designer/designer';

<DesignerI18nProvider locale="zh">
  <Designer ... />
</DesignerI18nProvider>
```
