# 组件属性矩阵

日期：2026-05-25

## 目标

统一组件属性面板的显示规则，避免每种组件都看到无关属性。属性按“常规、位置、内容、外观、文本、行为、数据、事件”组织。

## 分组规则

| 组件 | 内容 | 外观 | 文本 | 数据 | 专属属性 | 事件 | 渲染目标 |
| --- | --- | --- | --- | --- | --- | --- | --- |
| 文本 | 文本内容表达式 | 背景、边框、内边距 | 字体、字号、颜色、对齐、格式化 | 无 | 自动增大、自动缩小 | 支持 | 画布、预览、打印、PDF |
| 图片 | 图片地址/Base64 表达式 | 图片适配、背景 | 无 | 无 | 上传图片 | 支持 | 画布、预览、打印、PDF |
| 表格 | 单元格/列内容 | 表格边框、行高、表头 | 单元格阶段补齐 | JSON 表 | 行列、表头、表尾、跨页 | 支持 | 画布、预览、打印、PDF |
| 富文本 | 富文本内容 | 内容背景阶段补齐 | 内联编辑器 | 无 | 双击编辑、HTML 内容 | 支持 | 画布、预览、打印、PDF |
| 条码 | 条码内容表达式 | 背景/前景阶段补齐 | 显示文本阶段补齐 | 无 | 条码类型、显示文本 | 支持 | 画布、预览、打印、PDF |
| 复选框 | 选中表达式、标签文本 | 框样式阶段补齐 | 标签样式阶段补齐 | 无 | 勾选样式 | 支持 | 画布、预览、打印、PDF |
| 线条 | 无 | 无 | 无 | 无 | 颜色、宽度、样式、端点 | 支持 | 画布、预览、打印、PDF |
| 形状 | 无 | 无 | 无 | 无 | 形状类型、填充、边框 | 支持 | 画布、预览、打印、PDF |
| 面板 | 子组件容器 | 背景、边框、内边距 | 无 | 无 | 容器说明 | 支持 | 画布、预览、打印、PDF |
| 子报表 | 模板键、参数 | 无 | 无 | 参数表达式 | 嵌套模板 | 支持 | 画布、预览、打印、PDF |
| 页码 | 格式 | 无 | 字体、对齐 | 无 | 页码格式 | 支持 | 画布、预览、打印、PDF |
| 日期时间 | 格式 | 无 | 字体、对齐 | 无 | 日期时间格式 | 支持 | 画布、预览、打印、PDF |

## 当前阶段验收

- 每个组件只显示相关属性分组。
- 文本组件只使用“文本内容”作为主内容入口，不再暴露独立数据源/字段入口。
- 图片、线条等非文字组件不显示字体分组。
- 线条和形状显示自己的专属分组，不显示通用边框/外观分组。
- 表格显示表格分组，非表格组件不显示表格分组。
- 页码和日期时间显示字体分组和各自格式分组。
